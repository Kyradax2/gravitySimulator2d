gravitySimulator2D.py

language used:
	python 2.7.10

library used:
	pyGame (imported in the files through modules), but you need to install the package pygame with your package manager.

how to use:
	python gravitySimulator2D.py

author:
	Jérôme Paris

summary:
	gravitySimulator2D is a simple gravity simulator in a 2D environnement including randomly generated balls that collide with a board class object and the screen's boarders.
